# Usage
- please install PHP

if you installed Node.js, you execute follow code.

```
git clone https://github.com/tsukuba-pbl/16-4_Aggregate.git
cd 16-4_Aggregate
composer install
```

# Test

```
cd 16-4_Aggregate
phpunit --testdox ./test/specPhp
```
